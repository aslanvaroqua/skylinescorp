Page Title

If you don't need sliders and you want to build Simple Pages then you need Page Title and add it on your website following codes.

HTML File Structure

Page Title Normal 
<section class="uk-section uk-text-center uk-flex uk-flex-middle uk-flex-center uk-light uk-background-cover uk-background-top-center"
    data-overlay="0.4" style="background-color: #151515; background-image: url(assets/demo/elements-bg.jpg); height: 500px;">
    <div class="tw-page-title-container tw-element">
        <h1 class="tw-page-title uk-text-uppercase">Page Title</h1>
    </div>
    <div class="tw-breadcrumb-container uk-position-absolute uk-position-bottom-center tw-element">
        <a href="index.php" class="tw-breadrumb-child home">Home</a>
        <a href="#" class="tw-breadrumb-child">Breadcrumb</a>
        <a href="#" class="tw-breadrumb-child">Page Title</a>
    </div>
</section>

Note that height will controll the Height of the Page Title and data-overlay will work same like the Sliders. Please change the href to your website URL.

Parallax Page Title 
&lt;section class=&quot;uk-section uk-text-center uk-flex uk-flex-middle uk-flex-center uk-light tw-parallax&quot; data-stellar-vertical-offset=&quot;0&quot;
    data-stellar-background-ratio=&quot;0.5&quot; data-overlay=&quot;0.4&quot; style=&quot;background-color: #151515; background-image: url(assets/demo/slider/slider-blog-1.jpg); height: 600px;&quot;&gt;
    &lt;div class=&quot;tw-page-title-container tw-element&quot;&gt;
        &lt;h1 class=&quot;tw-page-title uk-text-uppercase&quot;&gt;{{ $element_name }}&lt;/h1&gt;
    &lt;/div&gt;
    &lt;div class=&quot;tw-breadcrumb-container uk-position-absolute uk-position-bottom-center tw-element&quot;&gt;
        &lt;a href=&quot;index.php&quot; class=&quot;tw-breadrumb-child home&quot;&gt;Home&lt;/a&gt;
        &lt;a href=&quot;#&quot; class=&quot;tw-breadrumb-child&quot;&gt;Elements&lt;/a&gt;
        &lt;a href=&quot;#&quot; class=&quot;tw-breadrumb-child&quot;&gt;{{ $element_name }}&lt;/a&gt;
    &lt;/div&gt;
&lt;/section&gt;

Only Difference is the data-stellar is inserted and also you need to insert Big Images. It means enought to parallax Height Image.
